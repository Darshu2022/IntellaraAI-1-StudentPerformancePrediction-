# 📚 Student Performance Prediction System

## 📌 Project Overview

This project predicts a student's **Exam Score** using Machine Learning techniques. It analyzes different academic and personal factors such as study hours, attendance, sleep hours, motivation level, internet access, previous scores, and more.

The goal is to compare multiple regression algorithms and identify the best-performing model for predicting student performance.

---

## 🎯 Objectives

- Load and explore the dataset
- Perform Exploratory Data Analysis (EDA)
- Clean and preprocess the data
- Create visualizations
- Train multiple Machine Learning models
- Compare model performance
- Build a prediction function
- Save the trained model

---

## 📂 Dataset

- **Dataset:** Student Performance Factors Dataset
- **Source:** Kaggle

Target Variable:

- **Exam_Score**

## 🤖 Machine Learning Models

The following regression models were trained:

1. Linear Regression
2. Decision Tree Regressor
3. Random Forest Regressor

---

## 📈 Model Evaluation Metrics

The models were evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

Random Forest provided the best performance among the tested models.
